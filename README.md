# E-Commerce Customer Feedback Analysis

## Overview
Analysis of customer feedback data from multiple sources (CSV, JSON, XML) 
for an e-commerce company to improve product quality and customer satisfaction.

## Project Structure
- `data/` — Raw data files (CSV, JSON, XML)
- `scripts/` — SQL scripts for database schema and queries
- `parsing/` — Python scripts for data parsing
- `reports/` — Final reports and presentation slides
- `images/` — ER diagram and visuals

## Tools & Technologies
- Python (csv, json, xml.etree)
- MySQL
- SQL (GROUP BY, JOIN, CASE statements)
- Pandas

## Key Analysis
- Top-rated products by average rating
- Common complaints using keyword search
- Customer sentiment classification (Positive/Neutral/Negative)
- Sales trends over time by category
